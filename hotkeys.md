# Terminal
| Description | Hotkey | Details|
| -------- | -------| ---|
| Find file by name | `find <dir> -name "<pattern>"` | 
| Grep basic | `grep -r -i -c <pattern> <dir> pattern` | -r = recurse, -i = case-insensitive, -c = number of cases |
